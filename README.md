# zedelivery
Repositório do projeto ZeDelivery com Cucumber, Capybara e Ruby.

## Como executar o projeto

* Importante ter o Ruby instalado (versão 2.5 ou superior)

### Instalar o bundle
`
gem install bundle
`

### Instalar as dependências do ruby
`
bundle install
`

### Executar localmente 
`
bundle exec cucumber
`
